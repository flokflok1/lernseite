# Compliance Evidence

**Purpose:** Evidence collection for regulatory compliance audits

## Directories

- `gdpr/` - GDPR (General Data Protection Regulation) evidence
- `iso27001/` - ISO 27001:2022 Information Security Management evidence
- `dsa/` - DSA (Digital Services Act) evidence
- `netzdg/` - NetzDG (German Network Enforcement Act) evidence
- `audits/` - External audit reports and certifications

## What to Store Here

### GDPR Evidence
- Data Processing Records (Art. 30)
- DPIA (Data Protection Impact Assessments) reports
- Consent management logs
- Data subject requests (access, erasure, portability)
- Breach notifications (72h reports)

### ISO 27001 Evidence
- ISMS documentation
- Risk assessments
- Internal audit reports
- Management review minutes
- Evidence of control implementation (Annex A)

### DSA Evidence
- Content moderation logs
- Transparency reports (quarterly)
- User reporting records
- Algorithm transparency documentation
- Crisis response logs (if VLOP)

### NetzDG Evidence
- Illegal content reports
- Response time tracking (24h/7d SLAs)
- Transparency reports (semi-annual: Jan 31, Jul 31)
- German representative contact info

## Security

⚠️ **IMPORTANT:** This directory contains sensitive audit evidence!
- Do NOT commit to public repos
- Add to `.gitignore`
- Encrypt if storing PII
- Access: Compliance Officer + Auditors only
